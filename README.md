# Daily Motivational App

A minimalist app that delivers daily motivational quotes and helps users track personal goals and mood.

## Features

- Daily curated motivational quotes
- Personal goal tracking (daily, weekly, monthly)
- Mood journaling with reflection prompts
- Streak counter for daily engagement
- Custom notification reminders
- Offline quote access
- Dark/Light mode themes
- Achievement system

## Installation

### Prerequisites

- Node.js (v16+)
- React Native development environment
- Android Studio or Xcode

### Setup

```bash
git clone https://github.com/yourusername/daily-motivational-app.git
cd daily-motivational-app
npm install
cd ios && pod install && cd ..  # iOS only
npm start
```

### Run

```bash
npm run ios     # iOS
npm run android # Android
```

## Tech Stack

- React Native
- TypeScript
- React Navigation
- Async Storage
- Zustand (state management)
- React Native Push Notification

## Project Structure

```
src/
├── components/    # UI components
├── screens/       # App screens
├── navigation/    # Navigation setup
├── services/      # Data services
├── store/         # State management
└── utils/         # Helper functions
```

## Usage

### Adding Custom Quotes

Edit `src/data/quotes.json`:

```json
{
  "id": "unique-id",
  "text": "Your quote here",
  "author": "Author Name",
  "category": "motivation"
}
```

### Goal Categories

- Personal Development
- Health & Fitness
- Career & Business
- Relationships
- Hobbies & Interests
